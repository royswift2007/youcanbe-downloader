## 3. 实际完成项

### 3.1 新增字段

新增位置：[youtube_models.py](/c:/Users/qinghua/Documents/youcanbe%20downloader/core/youtube_models.py)

- `subtitle_mode`
  - 用途：字幕模式，支持 `none/manual/auto/both`
  - 默认值：`none`
- `subtitle_langs`
  - 用途：字幕语言过滤（逗号分隔）
  - 默认值：空字符串
- `subtitle_format`
  - 用途：字幕格式（如 `srt`、`vtt`、`ass`）
  - 默认值：空字符串
- `embed_subs`
  - 用途：是否内嵌字幕
  - 默认值：`False`
- `write_subs`
  - 用途：是否写出外挂字幕
  - 默认值：`True`

### 3.2 builder 改动

改动位置：[ytdlp_builder.py](/c:/Users/qinghua/Documents/youcanbe%20downloader/core/ytdlp_builder.py)

- 新增字幕参数组装：
  - `manual` -> `--write-subs`
  - `auto` -> `--write-auto-subs`
  - `both` -> 两者同时写入
- 支持：
  - `--sub-langs`
  - `--sub-format`
  - `--embed-subs`
- 保留旧字段兼容：
  - `sub_lang` 仍可驱动旧逻辑映射到 `manual + embed` 路径

### 3.3 UI 改动
